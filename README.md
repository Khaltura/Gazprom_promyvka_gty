# Учёт промывочных материалов

Django-приложение для учёта промывочной жидкости и антифриза по компрессорным станциям.

## Возможности

- регистрация, вход и выход;
- ручное назначение КС пользователю через админ-панель;
- пользователь видит только свою КС, суперпользователь — все КС;
- приход, списание и перераспределение промывочной жидкости и антифриза;
- контроль доступного остатка;
- журнал промывок;
- баланс с группировкой по типам материалов;
- аналитика с графиками и фильтром по периоду;
- экспорт баланса, операций и промывок в Excel.

## Запуск на Windows

```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
python -m pip install -r requirements.txt
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```

Откройте `http://127.0.0.1:8000/`.

После регистрации назначьте пользователю КС в `/admin/` → «Профили пользователей».
